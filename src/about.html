About This Project

Overview

This project is a simple web server written in C. It is designed to serve static HTML files, such as this one, and handle basic HTTP requests. The server listens on port 8080 and can serve multiple pages, including a homepage and an about page.

How It Works

The server uses socket programming to accept incoming connections, parse HTTP requests, and serve the appropriate files. It supports basic file handling, such as reading HTML files and sending them as responses to clients. The server is lightweight and can be extended to support more advanced features.

Features

- Serves static HTML files.
- Handles basic HTTP GET requests.
- Supports multiple pages (e.g., homepage, about page).
- Lightweight and easy to extend.
